## AI Agent Learn Plan
- 主要目标：深入学习 AI Agent 开发和架构设计，具备企业对 AI 建设的能力要求
- 主要技能：AI Agent 模式学习，框架学习 langchain，langgraph、autogen，RAG，MCP、多 Agent等；从原理到应用
- 实践：Agent 练习，逐步进阶，比如单 Agent，到多 Agent，架构设计，RAG 本地搭建，云端搭建，工具调用等
- 形式：代码实际练习，文章输出
- 当前基础：有编程能力，对 Ai 有一定了解和基础使用，但没有更多实践
